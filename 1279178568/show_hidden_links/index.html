Showing links on hover

More script and css style : www.htmldrive.net

This page is created by Marcofolio and shows a pretty useful webdesign technique. There are four paragraphs located below. The first and the third are normal; The second and fourth have some extra CSS applied. At first, the paragraphs look link-free. But when the user hovers over the paragraph, the hidden links will be revealed.

This technique can be useful when you want to add links to certain paragraphs, but don't need a lot of attention from the user. Sadly, using only CSS, it isn't cross-browser compatible (read the article). That's where jQuery kicks in to make it work everywhere.
